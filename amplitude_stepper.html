What it does

Type a squared-amplitude expression (or import one from the editor) and it derives |M|² step by step — gamma-trace algebra via a Wick-contraction engine, contraction to Mandelstam variables (s, t, u), and the final result. No looked-up answers.

Entering an amplitude

  • Type into the Diagram 1 box, e.g. (e4/s2) * Tr[p1 gm p2 gn] * Tr[p3 gm p4 gn].
  • Solve ▶ or Ctrl+Enter to run.
  • Syntax ? (top bar) lists the full grammar: traces Tr[…], momenta p1…p8/k, gammas gm gn gr gs ga gb, scalars e4 gs2 m2, denominators /s /t /u, spinor bilinears.

Spacetime dimension dim d

  • Sets d in the trace algebra: γμγμ = d, the (d−4) dimensional-regularisation term, and the loop master integral Γ(n−d/2)/(4π)d/2.
  • Changing d re-solves live — coefficients and poles update (e.g. e⁺e⁻→μ⁺μ⁻ gives 8e⁴(t²+u²)/s² at d=4, 8e⁴(t²+u²+s²)/s² at d=6).

N-diagram interference

  • + Diagram adds an extra input box. Each extra box has its own ± sign toggle: − for identical-fermion exchange (Bhabha, Møller), + for bosonic / distinct diagrams. Solves |ΣMᵢ|² = Σ|Mᵢ|² + Σᵢ<ⱼ 2sᵢsⱼRe(MᵢMⱼ*) for any number of diagrams. Bhabha and Møller examples load automatically.

Import from Editor

  • In the editor: draw → iM ▶ → → Stepper. This opens (or refocuses) this page directly with the amplitude attached, adopts the editor's dimension d, and solves. Multiple connected components import as separate diagrams that add.
  • On a different laptop/browser, use the editor's ⇩ Export File button, then ⇧ Import File here instead.
  • Editing an input box switches it back to parsing your typed text.

Examples

Examples ▾ loads ready-made cases (QED ee→μμ, eμ→eμ, Compton, QCD, Yukawa, Bhabha, Møller, single traces, spinor bilinears) — the fastest way to see the format.
